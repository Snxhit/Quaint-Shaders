## Overview

This project was created in [Macondo](https://macondo.hackclub.com)!<br>

**Project Highlights**:

- Gives the game a grotesque but cozy feel.
- Revamps the classic Minecraft look.
- Adds shadows, tonemapping, edge highlighting, and a lot more.

---

## Features

- Grotesque yet cozy Minecraft visuals with a stylized, vibrant art direction
- Soft shadow rendering for smoother, more atmospheric lighting
- Time-of-day aware sunlight that changes throughout the day
- Rain-tinted lighting for a moodier look during storms
- Animated water waves with time-based motion
- Water & foliage waving toggles
- Multi-pass post-processing pipeline
- Several screen-space effects:
    - Shadows
    - Bloom
    - Fog
    - ACES Tonemapping
    - Edge detection (Grid & Contour)
    - Chromatic aberration
    - FXAA (Fast Approximate Anti-Aliasing)
    - SSAO (Screen Space Ambient Occlusion)
    - Each with their own threshold settings!
- Shader internals customized for each dimension
- Advanced lighting settings

## Usage

### Requires Iris! Supports Windows, MacOS & Linux!

- **Install from Modrinth**

   Find the Quaint Shaders Modrinth page [here](https://modrinth.com/shader/quaint-shaders)!

#### Or, alternatively

- **Install from GitHub Releases**

    Download the file [here](https://github.com/Snxhit/Quaint-Shaders/releases) and place it in your `/shaderpacks` folder.
